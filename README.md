# 자바 웹 프로그래밍 NestStep 스터디
## 스터디 진행 방식

각자 챕터에 주어진 과제를 해결합니다. 해결한 과제는 토요일 저녁 8시까지 저장소에 업로드합니다.

반드시 모든 내용을 완벽하게 구현할 필요가 없습니다. 점진적으로 리팩토링하여 기술에 대한 이해와 어떻게 해야 좋은 코드를 작성할지 고민하고 공유합니다.
